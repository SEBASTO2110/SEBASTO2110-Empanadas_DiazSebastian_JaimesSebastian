## Descripción del proyecto

Este programa permite administrar un catálogo de empanadas desde la consola.  
El sistema ofrece un menú interactivo donde el usuario puede **listar, agregar, editar y eliminar empanadas**.

La información se almacena en un archivo **JSON**, lo que permite guardar los datos incluso después de cerrar el programa.

---

## Funcionalidades

El sistema cuenta con las siguientes opciones:

1. **Listar empanadas**  
   Muestra todas las empanadas registradas.

2. **Agregar empanada**  
   Permite registrar una nueva empanada en el sistema.

3. **Editar empanada**  
   Permite modificar la información de una empanada existente.

4. **Eliminar empanada**  
   Permite eliminar una empanada del catálogo.

5. **Salir del programa**  
   Finaliza la ejecución del programa.
